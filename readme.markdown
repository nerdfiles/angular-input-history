# angular-input-history

A minihistory interface implementation in AngularJS for text inputs that 
makes use of the memento design pattern.

    <input
      data-history="{
        baseUrl: ‘’,
        iconUrl: {
          forward: ‘’,
          back: ‘’
        },
        search: true
      }"
    />
